# Example end-user service: A forestry climate service Harvester Seasons

With smartmet server as backend, yOu can build services for end-user showcasing the data produced (or used) in cryoscope project. harvesterseasons.com is a service for forestry operators, you can find the FMI Github repository harvesterseasons-smartmet, which contains the entire backend system supporting the service. On the service web page, you can click for more information to know of the service more details. 

The service has three main components: index for trafficability conditions, WMS layers, and time series graphs.
